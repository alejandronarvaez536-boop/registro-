# Sistema de Registro de Alumnos

## Descripción
El **Sistema de Registro de Alumnos** es una aplicación desarrollada para gestionar la información básica de los estudiantes de una institución educativa.  
Permite registrar, consultar y mostrar los datos de los alumnos de manera organizada.

Este sistema está diseñado como una práctica académica para aplicar conceptos de programación como **clases, arreglos de objetos y métodos**.

---

## Objetivos del Proyecto

### Objetivo General
Desarrollar un sistema que permita almacenar y gestionar la información de estudiantes.

### Objetivos Específicos
- Registrar alumnos en el sistema.
- Mostrar la información de los alumnos registrados.
- Calcular el promedio de calificaciones.
- Practicar programación orientada a objetos.

---

## Características del Sistema

El sistema permite:

- Registrar alumnos
- Mostrar datos de los alumnos
- Calcular promedio de calificaciones
- Almacenar información en arreglos de objetos

---

## Estructura de Datos

Cada alumno contiene la siguiente información:

| Campo | Tipo de dato | Descripción |
|------|--------------|-------------|
| Nombre | string | Nombre del estudiante |
| ID | int | Número de identificación |
| Calificación | float | Nota obtenida en el curso |

---

## Tecnologías Utilizadas

- Lenguaje: **C++**
- Paradigma: **Programación Orientada a Objetos**
- Entorno de desarrollo: **Visual Studio / CodeBlocks / Dev C++**

---

## Estructura del Programa

El programa contiene:

- Clase **Estudiante**
- Métodos para:
  - Registrar estudiante
  - Mostrar estudiante
  - Calcular promedio
- Arreglo de objetos para almacenar estudiantes

---

## Ejemplo de Funcionamiento

1. El usuario ejecuta el programa.
2. Ingresa los datos de los estudiantes.
3. El sistema almacena la información.
4. El sistema muestra los datos registrados.
5. Se calcula el promedio de las calificaciones.
